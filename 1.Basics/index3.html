Selenium Setup and Installation

Learn how to install Selenium WebDriver, browser drivers, and language bindings for test automation

Prerequisites

  • Java Development Kit (JDK)
  • Python (if using Python)
  • .NET SDK (if using C#)
  • Node.js (if using JavaScript)
  • Ruby (if using Ruby)
  • IDE (Eclipse, IntelliJ, VS Code)

WebDriver Setup

  • Download browser-specific drivers
  • ChromeDriver for Chrome
  • GeckoDriver for Firefox
  • Add drivers to system PATH
  • Or specify driver path in code
  • Use WebDriver Manager for automation

Language Bindings

  • Java: Add Selenium JARs to classpath
  • Python: pip install selenium
  • C#: Install Selenium.WebDriver NuGet
  • JavaScript: npm install selenium-webdriver
  • Ruby: gem install selenium-webdriver
  • Verify installation with simple test

Step-by-Step Installation Process

1

Install Java

Download and install JDK from Oracle's website

2

Set up IDE

Install Eclipse, IntelliJ, or VS Code with necessary plugins

3

Download Drivers

Get ChromeDriver, GeckoDriver, etc. from official sources

4

Add to PATH

Extract drivers and add their location to system PATH

5

Add Selenium Libraries

Add Selenium JAR files to your project or install via package manager

6

Verify Setup

Write and run a simple test to confirm everything works

Installation Commands for Different Languages

# Python
pip install selenium

# Java (Maven - add to pom.xml)
<dependency>
    <groupId>org.seleniumhq.selenium</groupId>
    <artifactId>selenium-java</artifactId>
    <version>4.10.0</version>
</dependency>

# JavaScript/Node.js
npm install selenium-webdriver

# C# (.NET Core CLI)
dotnet add package Selenium.WebDriver

# Ruby
gem install selenium-webdriver

Basic Selenium Test in Java

import org.openqa.selenium.WebDriver;
import org.openqa.selenium.chrome.ChromeDriver;

public class FirstTest {
    public static void main(String[] args) {
        // Set the path to chromedriver
        System.setProperty("webdriver.chrome.driver", "path/to/chromedriver");

        // Initialize the WebDriver
        WebDriver driver = new ChromeDriver();

        // Navigate to a website
        driver.get("https://www.google.com");

        // Get and print the page title
        String title = driver.getTitle();
        System.out.println("Page title is: " + title);

        // Close the browser
        driver.quit();
    }
}
