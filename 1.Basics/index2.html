Understanding Selenium

Learn about Selenium, its components, architecture, and how it works for web automation testing

Selenium WebDriver

  • Core component for browser automation
  • Provides programming interface
  • Controls browsers directly
  • Supports multiple languages
  • Cross-platform compatibility

Selenium IDE

  • Record and playback tool
  • Browser extension
  • No programming knowledge required
  • Creates quick test cases
  • Export tests to WebDriver code

Selenium Grid

  • Distributed test execution
  • Run tests in parallel
  • Cross-browser testing
  • Multiple environments
  • Saves execution time

Selenium Architecture

Selenium uses a client-server architecture where the client (test script) communicates with the browser-specific driver, which then controls the browser.

The JSON Wire Protocol facilitates communication between client libraries and browser drivers.

How Selenium Works

1

Test Script

You write a test script using Selenium client library in your preferred language

2

JSON Commands

Selenium converts your commands into JSON format and sends to browser driver

3

Browser Driver

The driver receives commands and executes them on the actual browser

4

Execution

Browser executes the commands and returns response back through the driver

Simple Selenium Code Example (Java)

import org.openqa.selenium.WebDriver;
import org.openqa.selenium.chrome.ChromeDriver;
import org.openqa.selenium.By;
import org.openqa.selenium.WebElement;

public class FirstSeleniumTest {
    public static void main(String[] args) {
        // Set the path to chromedriver
        System.setProperty("webdriver.chrome.driver", "path/to/chromedriver");

        // Initialize the WebDriver
        WebDriver driver = new ChromeDriver();

        // Navigate to a website
        driver.get("https://www.example.com");

        // Find an element and interact with it
        WebElement element = driver.findElement(By.id("someId"));
        element.click();
        element.sendKeys("Test input");

        // Verify page title
        String title = driver.getTitle();
        System.out.println("Page title is: " + title);

        // Close the browser
        driver.quit();
    }
}
